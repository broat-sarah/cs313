What is LocalFeed?

LocalFeed is a Java and JavaScript-based web app that allows user to find local webcam feeds at the click of a button. Simply use our search bar to select a location from Google Maps and designate a radius, then see all of the current webcam feeds near your address.


What can I use LocalFeed for?

  • Checking road conditions
  • Visiting any new cities- anywhere in the world
  • Checking out new landmarks

Future Enhancements

  • Report a webcam if it's down
  • Mobile application
  • Mobile device sync
